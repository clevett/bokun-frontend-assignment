# Frontend Assignment

Frontend dev project.

## Setup

Requires a .env file for api.

1. `npm install` in the root folder
2. Create a .env file in the root with key `NEXT_PUBLIC_API = "{demoAddress}/{assignedId}"`
3. Run in terminal `npm run dev`
